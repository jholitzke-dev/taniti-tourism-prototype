# Taniti Tourism Website Prototype

Interactive HTML prototype based on the Taniti tourism website design.

Open `index.html` in a browser to preview locally, or upload the folder/ZIP to Netlify to get a public URL.

## Navigation
- Home: `index.html`
- Where to Stay: `stay.html`
- Dining: `dining.html`
- Activities: `activities.html`
- Getting Around: `transport.html`
- Travel Tips / FAQ: `tips.html`
- Book Now: `book.html`
