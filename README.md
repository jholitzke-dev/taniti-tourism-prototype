

# Taniti Tourism Website Prototype

This project is an interactive tourism website prototype created for the fictional island of Taniti. The goal of the redesign is to improve the island’s outdated tourism website by creating a more organized, welcoming, and user-friendly experience for prospective visitors.

The prototype focuses on helping tourists quickly find important trip-planning information, including lodging, dining, activities, transportation, booking options, and frequently requested travel details. The design is based on the needs of a family vacation persona who wants to plan a safe, enjoyable, and organized trip to Taniti.

Key features of the prototype include clear navigation, dedicated pages for major travel categories, family-friendly activity information, transportation guidance, travel tips, FAQ content, and a central “Book Now” planning page. The site is designed to support a smoother user flow from exploring the island to planning the details of a visit.

This project demonstrates UX design planning, persona-based decision-making, low-fidelity prototyping, usability task development, and interactive web presentation through GitHub Pages.
